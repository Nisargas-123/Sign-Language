# Sign Language Recognition System

## Project Overview

This project is a **Machine Learning-based Sign Language Recognition System** that recognizes **English alphabet hand signs (A–Z)** using Computer Vision and Deep Learning. The system captures hand gestures from images or a webcam, processes them, and predicts the corresponding alphabet.

The project is designed to assist in understanding basic sign language alphabet recognition and demonstrates the application of Machine Learning in gesture classification.

---

## Features

- Recognizes English Sign Language Alphabets (A–Z)
- Machine Learning-based Gesture Classification
- Image Processing using OpenCV
- Deep Learning Model for Prediction
- Real-time Alphabet Detection
- User-friendly Interface
